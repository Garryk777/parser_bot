# 🤖 Telegram бот-парсер (MVP)

Бот для мониторинга сообщений в чатах по ключевым словам с возможностью подписки.

## 🚀 Функционал (в разработке)

- Мониторинг заданных чатов
- Фильтрация по ключевым словам
- Мгновенные уведомления
- Подписка (админ-панель)

## 🛠️ Стек

- Python 3.12
- aiogram 3.x
- Telethon
- SQLite
